# Patrones discursivos asociados a depresión y riesgo suicida en redes sociales: análisis comparado de siete países hispanohablantes mediante modelos de lenguaje 🧠📊

Bienvenido al repositorio oficial de nuestro Trabajo de Fin de Máster (TFM). Este proyecto investiga y compara cómo se expresa el sufrimiento psicológico y la salud mental en la red social X (anteriormente Twitter) a través de **7 países de habla hispana**.

**Grupo 8 - Autores del proyecto:**
* Ignasi Barnadas 
* Sílvia Bayés
* Micaela Beti
* Gabriela Martín

Mediante el uso de Procesamiento de Lenguaje Natural (NLP) y Modelos de Lenguaje de Gran Escala (LLMs), este estudio clasifica el riesgo emocional y extrae las temáticas principales del discurso digital.

## 🗂️ Estructura del Repositorio (Organizada por Países)

Para facilitar el análisis comparativo transcultural, este repositorio está estructurado geográficamente. Cada carpeta nacional es independiente y contiene su propio ecosistema de archivos: el *pipeline* de código (Notebooks de Python), los datasets anonimizados (CSV/JSON) y los reportes temáticos específicos.

* 🇦🇷 **Argentina/** -> Códigos de extracción, limpieza y datasets de Argentina.
* 🇨🇱 **Chile/** -> Códigos de extracción, limpieza y datasets de Chile.
* 🇨🇴 **Colombia/** -> Códigos de extracción, limpieza y datasets de Colombia.
* 🇪🇸 **España/** -> Códigos de extracción, limpieza y datasets de España.
* 🇲🇽 **México/** -> Códigos de extracción, limpieza y datasets de México.
* 🇵🇪 **Perú/** -> Códigos de extracción, limpieza y datasets de Perú.
* 🇻🇪 **Venezuela/** -> Códigos de extracción, limpieza y datasets de Venezuela.

## 📈 Visualización Global
* 📁 **dashboard/**: Contiene el archivo de Power BI (`.pbix`) global que unifica las métricas y los pesos relativos de los macrotemas de salud mental de todos los países.

## ⚙️ Tecnologías Utilizadas
* **Lenguaje:** Python 3.x (Pandas, JSON, Requests)
* **Inteligencia Artificial:** Google Generative AI (Gemini 2.5 Flash)
* **Visualización:** Microsoft Power BI

## ⚠️ Nota Ética y de Privacidad
Todos los datos presentes en este repositorio han sido rigurosamente anonimizados. Los identificadores de usuario reales, ubicaciones exactas y nombres propios han sido enmascarados (ej. `USER_3897376`, `{{LOCATION}}`) para cumplir con los estándares éticos de investigación psiquiátrica y de ciencia de datos.
